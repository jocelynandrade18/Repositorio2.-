# Repositorio2.-
Actividades realizadas en Inteligencia Artificial

## Contenidos
Dentro del repositorio se aprecian varias carpetas que parecen indicar distintos temas de redes neuronales o de aprendizaje automático:
1. CNN (Convolución Neural): Como la calsificación de ropa y dígitos a través de una imágen se predice lo que es.
2. McCulloch-Pitts: Haciendo referencia al modelo neuronal clásico de McCulloch-Pitts con operadores lógicos como NOT y OR 
3. Red Multicapa (MLP): Predicción de la popularidad de una canción, eficiencia del combustible, con operadore lógico XOR, clasificación del IRIS y clasificación de candidatos.
4. Perceptrón: Sencillo modelo de perceptrón, parte de IA básica con operadores lógicos como NOT y OR. 
5. RNN: Una neuronal recurrente que predice el precio del Bitcoin.

Estas carpetas sugieren que el repositorio agrupa actividades educativas y ejercicios relacionados con diversas arquitecturas de redes neuronales dentro del campo de la Inteligencia Artificial.

Los archivos dentro (notebooks .ipynb) servirían para demostrar la implementación y estudio de cada tema: Perceptrón, Multicapa, CNN, RNN, y un Modelo clásico de McCulloch-Pitts.
